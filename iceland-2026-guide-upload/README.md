# 2026 国庆冰岛攻略

这是一个纯静态网页攻略，入口文件是 `index.html`。

## 本地预览

```bash
python3 -m http.server 4203
```

打开：

```text
http://127.0.0.1:4203/
```

## 长期分享给朋友

推荐使用 GitHub Pages：

1. 在 GitHub 新建一个公开仓库，例如 `iceland-2026-guide`。
2. 上传本文件夹内所有文件：`index.html`、`script.js`、`styles.css`、`assets/`、`.nojekyll`。
3. 进入仓库 `Settings` → `Pages`。
4. `Source` 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 1-2 分钟，GitHub 会生成一个长期网址。

以后只要更新这些文件并推送到 GitHub，朋友刷新同一个网址就能看到最新版。
